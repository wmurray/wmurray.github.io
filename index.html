Welcome!

You've found me! I am Will Murray, a Boston-based web developer. This is my portfolio and the blog of my endeavors. If you'd like to know more, stop by the About page.

What I Do

My passion is front end web development. I prefer using clean, semantic markup and styles combined with unobtrusive JavaScript to make a smooth and enjoyable experience. With a backround in design, I aim to consider the user first when crafting the front end solutions for your website design.

My goal is to create clean, readable code that runs smoothly and efficiently, with the widest range of accessibility possible.

What I Use

Focusing on front end leaves a wide array of tools at my disposal. Here are some that I use regularly:

  • HTML5 and HAML

  • CSS3 and Sass for pre-processing.

  • JavaScript and jQuery.

  • Compass, Bourbon, Neat, Susy, and Singularity for mixins and grid systems.

  • BEM syntax for CSS class naming.

  • AMD JavaScript module loading pattern.

  • Grunt and Gulp task management.

  • Middleman static site generation.

In addition, I have worked with following technologies at a higher level (i.e.: not in a production environment, but my own experimentation):

  • ReactJS, AngularJS, and BackboneJS frameworks.

  • Basic Rails application setup.

  • Twig templating.

This list will continue to grow as I explore more of the growing world of web development.
