Work

Below are some examples of my web development projects. though they are functional, they are all subject to being updated/refactored/improved. This is not all of the work I have done, just a small smattering.

  • Patient Zero

    Final project for Front End Web Development class. Based on a dice game, the interface imitates a slot machine. It randomly generates three results (rolls 3 dice) and displays the results. Allows two players to use one browser window. A work in progress.

  • Rock, Paper, Scissors

    A game of rock paper scissors. Given the HTML markup and CSS styles, the task was to complete the JavaScript document to make the game functional. The game reports win/loss/tie, and keeps score.

  • Temperature Converter

    A JavaScript-powered temperature converter. Created from scratch - focus was on making the JavaScript work, not on making a stunning visual experience.

  • Startup Matchmaker

    This was a class assignment - given an image of a site layout, produce HTML markup and a stylesheet to produce the desired results. In addition, this example required adding a breakpoint to make the layout responsive, and easier to use on mobile devices.
